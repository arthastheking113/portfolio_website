My Web Development Journey

  • Duy Lan Le
  • Web Developer
  • 1

Every Company need a software engineer to build and maintain their website for their bussines. At the first time, I didn't have the fun to work with html, css, and javascript because it seem to be too many thing to learn to me. However, I get used to working with Python and Data for a while. I built some projects by using Python, my biggest Python project is a Auto Trading Bot.

Even though I get used to working with Python for a long time, but I still decided to learn about web development because I want to quit my low-paid salary I'm working at for 2 years.

So i decided to attend a coding bootcamp and hope that I can find a better job for my future. I think I'm a good problem solver because I solve a lot of algorithm problems when working with Python. I think that language is not a problem, I can learn it and write down immediately.

When learning web development, I think it is not too hard like the first time I saw it. I did a lot of practice and working on many projects. After all, I gain a lot of experience and pretty confidence with it. In coding bootcamp, I learn not only about html, css, and javascript, but also about C# and working with Xamarin to build app on Anroid and IOS device, which can lead me to have more chances to have a job in the future.
