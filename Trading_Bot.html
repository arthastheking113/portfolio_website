Python Auto Trading Bot

  • Duy Lan Le
  • Web Developer
  • 1

Stock trading is a way to earn money, so I always wonder why I haven't tried to build an auto trading bot using Python. I'm using at that time because I get used to working with Python for a long time, and have many experience with it.

My trading algorithm is about buy and sell signal. In order to have that buy and sell signal, I used many candle stick modles, calculating RSI, golden, and death cross. I choose robinhood to be my broker because they have their own API and they don't have transistion's fee. And, i choose yahoo finance to get data from because they have a free API to get data and they don't limit how fast I took their data.

I wrote about 1000 lines of code to get and analyze data. There are hundreds of error appear when writing it and I think half of my time is about fixing bug and error. This project took me around 2 week to finish it, and now i'm still trying to simulate it to make sure if it really can earn money.
